# 🚺 empowHER – Woman's Companion & Safety App

An intelligent mobile application designed to enhance women's safety by providing real-time assistance, emergency alerts, and location-based security features.

---

## 📌 Project Overview

**empowHER** is built to ensure the safety and security of women by leveraging modern technologies like speech recognition, geolocation, and real-time communication. The app detects distress situations and automatically triggers emergency actions, making it a reliable companion during critical moments.

---

## 🛠️ Technologies Used

* Android Studio
* Java
* XML
* Firebase
* Google Maps API
* Geofencing API
* Speech Recognition API

---

## ✨ Features

### 🚨 Real-Time Safety Assistance

* Detects distress through voice recognition
* Automatically triggers SOS alerts

---

### 📍 Live Location Sharing

* Shares real-time location with:

  * Emergency contacts
  * Authorities
* Ensures quick response during emergencies

---

### 🗺️ Safe Zone Mapping

* Displays nearby safe zones using Google Maps
* Uses Geofencing API for location-based alerts

---

### 🎤 Hands-Free Emergency Activation

* Voice-based activation for emergency situations
* Enables quick action without manual interaction

---

### 👤 Contact & Profile Management

* Add and manage emergency contacts
* Store personal safety details securely

---

## 🚀 How to Run the Project

1. Open the project in **Android Studio**
2. Connect an Android device or start an emulator
3. Configure Firebase (if required)
4. Add API keys for:

   * Google Maps
   * Geofencing
   * Speech Recognition
5. Build and run the application
